# Eco Straws Vietnam – Giao diện mẫu (Bootstrap)

Mục tiêu: Dựng lại **giao diện/luồng nội dung** tương tự cấu trúc trang Eco Straws Vietnam bằng **HTML + Bootstrap 5 + jQuery**, **không sao chép** nội dung/hình ảnh gốc (tôn trọng bản quyền). Bạn chỉ cần thay thế văn bản, hình ảnh, liên hệ, bản đồ… theo thực tế.

## Cấu trúc
- `index.html` – Trang chủ (Hero, Sản phẩm, Quy trình, Chứng chỉ, CTA).
- `products.html` – Danh mục & bộ lọc sản phẩm (không dùng thư viện ngoài).
- `about.html` – Giới thiệu, sứ mệnh, giá trị cốt lõi.
- `factory.html` – Năng lực nhà máy + bản đồ nhúng.
- `contact.html` – Form liên hệ + info + bản đồ.
- `assets/css/styles.css` – Tuỳ biến giao diện.
- `assets/js/main.js` – Hiệu ứng nav, lọc sản phẩm, validate form, smooth scroll.

## CDN
- Bootstrap 5.3.3, Bootstrap Icons 1.11.3, jQuery 3.7.1 (thay đổi theo nhu cầu).

## Tuỳ biến nhanh
1. Tìm kiếm từ khoá `*mẫu*` trong các trang để thay nội dung bằng thông tin thật.
2. Đổi màu nhận diện trong `:root` (CSS).
3. Thay bản đồ `iframe` bằng toạ độ thực tế.
4. Kết nối backend/email thật cho form (hiện tại chỉ mô phỏng, hiển thị toast).

## Ghi chú bản quyền
- Không dùng logo, ảnh, hoặc đoạn văn sao chép từ website gốc.
- Mẫu này chỉ tái hiện **bố cục** và **loại nội dung** để bạn chỉnh sửa.
